# 🌸 Floka New — Digital Agency One Page Website

A modern, animated, and fully responsive **digital agency landing page** inspired by the Floka creative agency style.  
This project is built as a static front-end website with rich UI sections, smooth animations, portfolio layouts, service blocks, team showcase, FAQs, testimonials, and contact form design.

---

## 📌 Project Overview

**Floka New** is a creative agency portfolio website designed for presenting digital services in a professional and attractive way.  
The website focuses on clean visual design, smooth scrolling experience, modern typography, interactive sections, and strong brand presentation.

It is suitable for:

- Digital agency website
- UI/UX portfolio
- Web development agency
- Creative studio landing page
- Business portfolio website
- One-page company profile

---

## ✨ Main Features

- 🎨 Modern digital agency landing page design
- 📱 Fully responsive layout for desktop, tablet, and mobile
- 🧭 Header navigation with multi-level menu style
- 🚀 Animated hero section with creative agency branding
- 🎥 Video/reel section included
- 🖼️ Portfolio showcase section
- 🛠️ Services and expertise section
- 📊 Animated counters and fun facts
- 💬 Client feedback/testimonial section
- 📩 Contact and project inquiry form design
- 🏆 Awards and achievement section
- 👥 Team member showcase
- ❓ FAQ accordion section
- 📰 Blog/insights section
- 🎯 Smooth animations and interactive UI effects
